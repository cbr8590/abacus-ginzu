# PitchBook Data

Place PitchBook or other comp data here for future Gemini context integration.

**Supported formats:** CSV, PDF, Excel (.xlsx, .xls), TXT, Markdown

Examples of useful data:
- Sector benchmarks and industry averages
- Stage-appropriate metrics (seed, Series A, growth)
- Returns for comparable companies
- Typical revenue growth by sector/stage
- Operating margin benchmarks
